Entendo Comunicação Client x Server

(Voltar)

Neste módulo aprendemos sobre a história da web e somos apresentados a grandes nomes da tecnologia, como por exemplo o Tim Berners-Lee que inventou o "WWW" que chegou a receber o título de cavaleiro pela Rainha Elizabeth II. Entendemos também algumas diferenças que as aplicações webs e os aplicativos de dispositivos móveis tem, também a diferença entre Server x Clients, o funcionamento e como a informação pode trafegar desde o client até o banco de dados (DataBase).

Introdução ao HTML na Prática

(Voltar)

Aqui vemos ferramentas utilizadas como a IDE VSCode, que foi a IDE que utilizamos, vimos também algumas tags importantes para formatação de texto e para a organização, assim como também vimos os atributos que certas tags tem, que nos ajudam ainda mais com a organização/formatação, além de que as listas e os links podem se ligar para nos levar a links dentro do próprio site ou até mesmo fora.

Foram-nos introduzidas algumas tags: "h1 - h6" para ser usada em títulos e subtítulo. No mesmo seguimento a tag "hr" que é parecida com a "br", mas a "hr" quando usada traça uma linha e também adiciona mais margem. A taga para link "a", para parágrafo a tag "p", para adicionar um texto menor a tag "small", a tag "blockquote" que serve para destacar um texto como uma sitação, as tags "sub" e "sup" que servem para colocar o texto uma linha a baixo ou uma linha cima como por exemplo para 2 elevado a 2 (22), as tags para destacamento de texto como a "i" para itálico, a "strong" para negrito, a tag "u" para o sublinhado e também a tag "mark" para "passar um marca-texto".
